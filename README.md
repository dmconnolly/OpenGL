# OpenGL

Application created to showcase a variety of graphics techniques including : -
- Camera controls and movement
- Dynamic 3D terrain generation
- Dynamic cloud generation
- Shadows
- Tessellation
- Oren-Nayar and Blinn-Phong shaders
